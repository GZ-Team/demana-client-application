# demana-client-application

An Electron application built with Vue and TypeScript.

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

or

```bash
$ pnpm dev:hot-reload
```

### Build

```bash
# For windows
$ pnpm build:windows

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
